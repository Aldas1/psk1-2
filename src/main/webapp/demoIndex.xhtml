Jakarta EE Feature Demonstrations

Featured Demonstrations

This application implements various Jakarta EE features. The following pages demonstrate these features in action:

1. Optimistic Locking

Demonstrates OptimisticLockException handling and recovery strategies.

Key Points:

  • Transaction behavior during OptimisticLockException
  • EntityManager state after OptimisticLockException
  • Proper entity recovery and saving

2. Asynchronous Communication

Demonstrates asynchronous method calls and transaction behavior.

Key Points:

  • Async methods cannot join caller's transaction
  • Async methods cannot use @RequestScoped EntityManager
  • Future<T> return type for result handling

3. CDI Glass-box Extensibility

Demonstrates various CDI extension mechanisms.

Key Points:

  • CDI Alternatives with beans.xml configuration
  • CDI Specialization for bean replacement
  • CDI Interceptors for method interception
  • CDI Decorators for behavior enhancement

4. RESTful Services

Demonstrates REST API implementation with JAX-RS.

Key Points:

  • GET, POST, PUT, DELETE operations
  • Proper response handling
  • JSON serialization/deserialization
